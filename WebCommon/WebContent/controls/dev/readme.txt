最近更新
================================================================
2010.10.13  正式版更新 6.1.0.0， 
增加大单元格分页，本功能可以实现单个单元格中，有多个，多层html元素的自然分页（不断纵横边框线，不在文字中间断行），即使单元格内容由word转换而来亦可。


2010.08.19  正式版更新 5.7.0.0， 
增加 web 页面 和 MS Word,Excel,PowerPoint,PDF 文档的混合打印功能，本功能可以实现将非HTML文档作为附件打印的功能。  
增加pageFrom,pageTo属性，实现打印指定页功能。 

2010.08.09  正式版更新 5.6.0.0， 
增加打印 MS Word,Excel,PowerPoint,PDF 文档功能，利用本功能可以让用户打印这些非HTML附件，或用这些类型文件做模板的报表。 

2010.07.15  正式版更新 5.5.0.0，上市以来最稳定版本 !!! 
经过 Windows 2000/XP/2003/Vista/7+IE5.5/IE6/IE7/IE8全面测试,解决报 '请确认打印机已经正常连接!' 等bug。

2010.06.17  正式版更新 5.4.0.0 
增加页码分组功能。该功能允许用户在一次打印任务中，将一些页自动归为一组，每组页号分别计算。  
增加页内多表自动分页功能。该功能可以在一个div中包含多个表格，并使每个表格遇到分页时，都不会出现在行中间分页的情况   
增加预览时提示生成功能。该功能在预览较多页时，可弹出正在生成页面提示框. 


2010.04.28  正式版更新 5.2.0.0 
增加纵横打印功能。该功能允许在一次打印任务中，设置有些页纵向打印，有些页横向打印，这使你可以根据打印内容的特点，分别选择各个页面打印方向。 

2010.03.03  正式版更新 5.1.0.0 
增加套打位置微调信息保存在服务端功能。本功能的目的是允许某个用户(往往是管理员)在运行时来集中设置套打位置。以前的可视化设计套打功能只将位置信息保存到本地注册表，所以用户的位置微调不会影响到其他用户，而位置信息保存到服务端，则任何用户的位置微调将影响到所有其他客户。 
 
2009.12.29  NPAPI 插件发布，支持 firefox、chrome、safari 浏览器。 
jatoolsPrinter又添新成员，NPAPI 插件1.0正式发布，使jatoolsPrinter可以支持除IE外的大多数流行浏览器, 如 firefox、chrome、safari。 使您的web打印应用可以扩展到目前市场上95%以上的浏览器。

2009.12.08  正式版更新 5.0.0.1 
改进打印的启动速度。与前版本比较，启动速度大约快5倍以上。
增加双面打印功能。对于支持自动双面打印的打印机，你可以使用duplex属性来控制双面打印。


2009.11.03  正式版更新 4.3.2.0

增加打印页数事件监听，使用户可以根据打印的页数    


2009.07.18  正式版更新 4.3.0.0 
增加打印任务监听功能，使用户可以获取打印状态。 
增加零边距功能，使用户可以贴边打印。 


2009.05.18  正式版更新 4.2.0.0，增加套打的可视化设计功能 ，使杰表打印控件成为融设计、打印、预览于一体的web打印工具。 

在以前的版本中，如果用户想设计套打模板，必须通过网页设计工具做好网页，然后，放到杰表控件中来打印，如果发现打印内容的位置有偏差，需要再在网页设计工具中调整，然后再放到控件中，如此反复多次，效 率极低。现在有了可视化设计功能，可以使用户在控件中，就能调整位置，不仅如此，由于控件中指定位置，是在真实的、可视化的打印环境中，使用户不必考虑边距的问题，做到完全的所见即所得，可大大提升套打设计的效率。更多内容 ...

另外，我们也发现，即使是同一个套打网页，在不同的打印机中，打印出来的相对位置，也有差别，有的打印机打印出来的内容凑得紧些，有的则离得开些，这种偏差不能通过设置打印机边距的方式进行调整。借助可视化设计功能，使用户可以根据自己的打印机状况，自行调整打印内容的位置，并保存到注册表，下次打印时，自动根据自设的位置进行打印。这对有大量前台打印，并有不同类型打印机打印同一种票据的场合，非常实用。更多内容 ...

2009.02.05  正式版更新 4.0.0.0，增加页脚页眉，自动分页等功能 。
